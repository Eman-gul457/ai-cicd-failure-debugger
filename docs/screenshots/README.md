# Portfolio Screenshots

The screenshots in this directory are sanitized for public portfolio use.

Sensitive values have been redacted where needed, including:

- n8n webhook URLs
- IP addresses and forwarding header values
- secrets, tokens, authorization values, and shared secret headers
- unrelated personal UI elements when they were not needed for the demo

Raw or private capture files should not be committed to this repository. Use the following ignored locations and patterns for any local-only assets:

- `docs/screenshots/raw/`
- `private/`
- `*.secret.png`
